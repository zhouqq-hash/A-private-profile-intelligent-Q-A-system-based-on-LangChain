# Demo 演示指南

本目录包含用于演示"私人资料智能问答系统"的示例文档。

## 快速体验（3 步）

### 第 1 步：启动应用

```bash
streamlit run app.py
```

### 第 2 步：上传演示文档

1. 在浏览器打开 `http://localhost:8501`
2. 点击侧边栏「Browse files」按钮
3. 选择 `demo/sample_resume.md`
4. 点击「🔨 索引文件」按钮

### 第 3 步：开始提问

在聊天框输入以下示例问题：

| 场景 | 示例问题 |
|------|----------|
| 项目梳理 | "我做过哪些项目？" |
| 技能提取 | "列出我的所有技能" |
| 指代追问 | "第一个项目用了什么技术栈？" |
| 岗位推荐 | "根据我的经历，适合投递什么岗位？" |
| 自我介绍 | "帮我整理一段 1 分钟的自我介绍" |
| 亮点挖掘 | "我的项目中最有技术含量的是什么？" |
| 实习匹配 | "我的 XX 实习经历和哪些岗位要求匹配？" |
| 简历优化 | "我简历中哪些技能描述可以更量化？" |

## 预期效果

- 系统会根据 `sample_resume.md` 的内容回答
- 每个回答会标注 `[来源: sample_resume.md]`
- 支持多轮追问（如"第一个项目"指代前文提到的项目）
- 对于文档中没有的信息（如"我会弹钢琴吗？"），系统会明确拒绝回答

## 自定义文档

你也可以上传自己的文档：

- **个人简历**（PDF/Word/Markdown）
- **项目报告**（PDF/Word）
- **课程笔记**（TXT/Markdown）
- **实习总结**（Word/PDF）

支持同时上传多个文档，系统会自动合并索引。
